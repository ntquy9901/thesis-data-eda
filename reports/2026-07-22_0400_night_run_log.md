## Key Findings

1. **News signal is real but very weak.** Best ΔR² ≈ +0.0007 at pk_t+1 from basic dual-group embeddings.
2. **Simplicity wins.** Basic 91-feature dual-group set outperforms all enhanced versions (EWMA, multi-EWMA, novelty, dispersion).
3. **More features = overfitting.** 493-feature "full" set tanks R² by -0.04 to -0.23 depending on horizon.
4. **Regime dependence.** News only helps in high-volatility regimes (ΔR² = +0.0012). In low volatility, it hurts.
5. **Few tickers benefit.** Only 3-7/30 tickers show positive news contribution in per-ticker evaluation.
6. **Residual model fails.** Two-stage HAR residual approach performs worse than direct prediction.
7. **OOS signal vanishes.** On held-out 2026 data, news features slightly hurt (ΔR² = -0.0014).
